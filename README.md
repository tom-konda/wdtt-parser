## これは何？
駅時刻表の作成ソフト WinDIATimetable で出力される、WDTT 形式のファイルを解析して JavaScript
オブジェクトに変換できるパーサです。

## 対応ブラウザ
- ES 2015+ に対応しているブラウザ

## 対応 Node.js バージョン

* 最新の LTS

## ライセンス
ライセンスは MIT になります。

## ビルドの仕方

1. `git clone https://github.com/tom-konda/wdtt_parser.git` を実行し、レポジトリをクローンします
2. `cd wdtt_parser` でカレントディレクトリを変更します
3. `npm install` を実行します
4. `npm run build` を実行します
